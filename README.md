# ÉcoGestion STMG

Plateforme gratuite de révisions pour les élèves de Première et Terminale STMG.

📖 Cours complets · 🎯 QCM interactifs · ⚡ Fiches Flash · 📊 Simulateur de note · 📄 Sujets de bac

## Stack

Astro 5 · React 19 · Tailwind CSS · Framer Motion · Netlify

## Développement

```bash
npm install
npm run dev     # → localhost:4321
npm run build   # → dist/
```

## Déployer

Push sur GitHub → connecter à Netlify → build command `npm run build`, publish `dist`.
